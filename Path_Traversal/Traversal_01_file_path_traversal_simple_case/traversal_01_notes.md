# Lab: File path traversal, simple case

### Instructions

This lab contains a path traversal vulnerability in the display of product images.

To solve the lab, retrieve the contents of the `/etc/passwd` file. 

### Solution

Clicking on an item in the shop and inspecting the page source we find an interesting parameter:

```html
<h3>Snow Delivered To Your Door</h3>
<img src="/resources/images/rating1.png">
<div id="price">$42.00</div>
<img src="/image?filename=21.jpg">
```

The image source is `/image?filename=` which warrants testing.

Capturing this in Burp, we can test for directory traversal in the `filename` parameter and access the `/etc/passwd` file with:

